f. Определить состав информации для кеширования на уровне приложения пользователя, API Gateway, уровня бизнес-логики и уровня репозитория. Список

1. Уровень приложения пользователя:
  - вход в приложение/веб-сервиса пользователя
  - информация о роботе-пылесосе (модель, номер)
  - тема интерфейса приложения/веб-сервиса
  - координаты расположения робота в пространстве
  - координаты точки базы подзарядки
  - последний режим работы робота

2. Уровень API Gateway
  - аккаунт пользователя
  - регистрационные данные робота

3. Уровень бизнес-логики
  - маршруты уборки помещений
  - режим работы робота
  - расписание работы робота

4. Уровень репозитория
  - аккаунт пользователя
  - регистрационные данные робота
